## üîç Search Strategy

### Multi-Engine Approach

Use ALL search engines for comprehensive coverage:

- **Google** - Broad coverage, recent content priority
- **Bing** - Good for video and image results
- **DuckDuckGo** - Less filtered results, privacy-focused
- **Brave Search** - Independent index, less manipulation

### Keyword Variations

Try multiple phrasings:

```
"Mendix [topic]"
"Mendix [version] [topic]"
"Mendix best practice [topic]"
"Mendix tutorial [topic]"
"Mendix example [topic]"
"How to [action] in Mendix"
"Mendix vs [alternative] [topic]"
```

### Time-Based Searches

- **Recent content**: Last year (for latest features, Mendix 11)
- **Historical content**: All time (for established patterns, proven approaches)
- Filter by date ranges to find version-specific information

### Source Verification

- Cross-reference information from multiple sources
- Prioritize recent content for version-specific features
- Check author credentials (MVPs, Mendix employees, experienced developers)
- Look for community validation (upvotes, comments, shares)
